# ja-poster
以下のヘッダーをqmdファイルで利用する。

```
---
title: 
format:
  ja-poster-typst:
    output-dir: "_output/main"
    preview: true
    keep-typst: false
    # output-ext: ".typ"
    poster-authors: 
    poster-banner: "_extensions/ja-poster/banner-img/basin-of-attraction6.png"
    footer-left: 
    footer-center: ""
    footer-right: ""
    bibliography: "reference.bib"
    csl: "apa.csl"
    suppress-bibliography: false
---
```
